# SQL
SQL queries for data cleaning, analysis, and reporting
This repository contains SQL scripts I’ve written as a data analyst to solve business problems. The queries focus on data extraction, cleaning, joining multiple tables, and creating summary views for reporting. Each file tackles a specific question like tracking sales trends, measuring customer complaints, or analyzing employee attrition. The goal is to show how SQL turns raw database tables into analysis-ready data for tools like Tableau and Power BI.
